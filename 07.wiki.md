## 위키(Wiki)
* GitHub Wiki는 GitHub 저장소의 일부로, 프로젝트 팀원들이 프로젝트와 관련된 정보를 쉽게 공유하고 문서화할 수 있는 플랫폼을 말하며 Wiki 문서는 마크다운으로 쉽게 작성이 가능

* Wiki를 통한 문서화는 프로젝트의 지식을 쉽게 공유하고 기록할 수 있도록 도움
